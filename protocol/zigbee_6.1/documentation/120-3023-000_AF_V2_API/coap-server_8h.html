void emberAfPluginCoapServerCreateDevice ( uint16_t  deviceIndex)

Creates a listener port based on the device table index.

When a new device joins the gateway, it will be placed into the device table at a unique index point. This method will open up the appropriate COAP port on which to listen for traffic.

Parameters
deviceIndexIndex of the device to create.
void emberAfPluginCoapServerRemoveDevice ( uint16_t  deviceIndex)

Remove a listener port based on the device table index.

When a device is removed from the device table, we need to also close the corresponding COAP port.

Parameters
deviceIndexIndex of the device to remove.

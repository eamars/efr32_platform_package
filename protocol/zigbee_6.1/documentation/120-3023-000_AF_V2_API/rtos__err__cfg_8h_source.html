49  * DEF_ENABLED, a string containing the file name and line number where the error has been

57  * For example, if set to DEF_ENABLED, it would be possible to print RTOS_ERR_NONE or
58  * RTOS_ERR_INVALID_ARG as a string instead of printing the numerical value associated,
59  * which would be 0 for RTOS_ERR_NONE and higher than 0 for RTOS_ERR_INVALID_ARG.
